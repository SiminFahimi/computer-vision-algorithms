# Computer Vision Algorithms in Python

This repository contains implementations of classic **computer vision algorithms**.
The focus is on **algorithmic understanding**, not library black-box usage.

## Implemented Algorithms

### 1. Canny Edge Detector

* Gaussian smoothing
* Gradient computation
* Non-maximum suppression
* Double thresholding
* Edge tracking by hysteresis

### 2. Harris Corner Detector

* Image gradient computation
* Structure tensor (second-moment matrix)
* Harris response function
* Corner detection using thresholding

### 3. Laplacian Pyramid

* Gaussian pyramid construction
* Down-sampling and up-sampling
* Laplacian level computation
* Multi-scale image representation

## Supporting Modules

* **Gaussian Kernel Generator**
  `kernel_create.py`

* **Test & Visualization Script**
  `output.py`

* **Sample Image**
  `lena_color.jpg`

## Requirements

* Python 3.10+
* NumPy
* OpenCV (cv2)
* Matplotlib

## Project Goals

* Understand core computer vision algorithms mathematically and programmatically
* Avoid high-level CV abstractions (no OpenCV Canny/Harris shortcuts)
